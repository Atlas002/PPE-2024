# INFO CLIENT 


## EMULATION
Pour run émuler l'app sur son téléphone , il faut que Expo Go  et le localhost soient sur le même wifi puis run la commande suivante :

```bash
   npx expo start
```


## INSTALLATION DE PACKAGE

```bash
   npx expo customize
``` 
